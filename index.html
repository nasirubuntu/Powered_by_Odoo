Manual Currency Rate Exchange

The Manual Currency Rate Exchange module allows users to manually set custom exchange rates for sales, invoice, register payments, Purchase, Bills, Register Payments, Invoicing > Customer > Payments and Invoicing > Vendors > Payments. It ensures accurate multi-currency accounting by overriding system rates, maintaining precise journal entries, and supporting real-world currency scenarios with full traceability.
